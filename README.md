# React Learning Projects (Scrimba)

## Description
This repository contains a collection of projects developed while studying **React** through the **Scrimba React Course**.  
Throughout the course, I complete interactive coding challenges and build practical projects designed to strengthen core React concepts and real-world front-end development skills.

The repository documents my progress while learning React, focusing on component-based architecture, dynamic interfaces, and modern JavaScript practices.

---

## About the Project
The Scrimba React course includes **170+ interactive coding challenges** and **six hands-on projects** designed to help students learn React by building real applications.

Instead of only watching lectures, the course emphasizes **active coding and project-based learning**, allowing students to progressively develop the confidence and technical skills required to build modern React applications.

During the course, the following core topics are covered:

- React Components
- JSX
- Styling in React
- Props
- State Management
- Event Handling
- Conditional Rendering
- Data Fetching
- Side Effects
- Controlled Components
- Accessibility and Optimization

Concepts covered:
- React components
- JSX syntax
- Styling in React

### 2. Data-Driven React — Travel Journal
This section introduces **dynamic rendering in React** using reusable components and external data.

Concepts covered:
- Reusable components
- Props
- Data-driven UI
- Mapping arrays to components
- Static assets handling

### 3. React State — Chef Claude

Concepts covered:
